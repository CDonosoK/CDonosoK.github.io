Projects

ATOR - INRIA

September 2024 - Present

ATOR, Asistive Teleoperation Of Robots, is a project that aims to develop a teleoperation system for bimanual robots, which allows the operator to control the robot in a more intuitive way. In this way, the operator can control the robot but also be asisted by the system in complex task. I develop the teleoperation system that used a VR System to control the robot and also develop the UI to visualize all the robot information in real time.

QG Scout - SK Godelius

January 2023 - February 2024

QG Scout is an autonomous mobile robot that is capable of navigating autonomously in outdoor environments, avoiding obstacles and following a predefined path while do the sampling in points of interest. I develop the navigation system that allows the robot to navigate in a safe and efficient way using my own stack which is based on ROS1. I also develop the teleoperation interface to control the robot remotely and visualize the robot information in real time.

CG Scout - SK Godelius

January 2023 - July 2024

CG Scout is an autonomous mobile robot for conveyor belt inspection. The robot collects images, thermal images and audio data from the conveyor belt, while detecting anomalies and defects in the belt, processing the data in real time and sending the results to the cloud. I design and develop the arquitecture of the software that allows the robot to process the data in real time and send it to the cloud. I also develop the navigation system that allows the robot to navigate in a safe and efficient way using my own stack which is based on ROS1. I also develop the teleoperation interface to control the robot remotely and visualize the robot information in real time.

El Caleuche

June 2021 - July 2022

El Caleuche it's a robotic catamaran that is being developed by RoboTe-UC, which aims to create an autonomous catamaran that meets the 10 challenges proposed by RobotX for the competition to be held in Australia'22. Within the project, I was in charge of developing the communication systems, as well as the system that will control the Caleuche. For this, I used ROS and developed a stack that allows the robot to navigate autonomously and communicate with the rest of the systems. I also developed the teleoperation interface to control the robot remotely and visualize the robot information in real time.

Thesis - RoboCop

February 2022 - December 2022

On my thesis, I develop a 3D SLAM Algorithm that allows the robot map and navigate autonomously in an unknown environment while detecting prohibited areas and obstacles. The algorithm is based on the use of LIDAR and depth cameras, which allows the robot to create a 3D map of the environment and navigate in a safe and efficient way. I also develop a small platform to test the algorithm.

Sunny

March 2021 - September 2021

Sunny is a robotic dog equipped with sensors and a autonomous navigation system, which allows it to navigate safely and autonomously. It is designed to accompany our elderly and in case of emergency make calls or send messages to emergency contacts. The project is currently on hold.

DoMa

March 2020 - December 2020

DoMa is a virtual assistant that allows users to perform household tasks through voice commands, such as controlling lights, curtains, setting alarms, playing music, etc. DoMa was selected by Semillero USM to participate in the Latin American entrepreneurship festival INCmty 2020, where it was among the top 12 finalists.
